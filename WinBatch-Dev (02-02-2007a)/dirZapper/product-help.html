dirZapper v1.0 for Windows 95/98/NT

Command Line Options:
Parm # Option Description
1 c:\temp fully qualified path such as c:\temp; this may be "any" path name desired - Be Careful what pathname you choose to use especially when using the -d command line option.
2 -s Scan all files in all subdirectories and total the size of each file
2 -d Scan all files in all subdirectories and DELETE each file including READ-ONLY files
3 -v Verbose Output - generally means you'll see a pop-up window during processing (not used and not required with -s)
3 -t Terse Output - generally means you won't see anything during processing other than an icon on the task tray (not used and not required with -s)
Command Line Example #1: dirZapper c:\temp -s (Scan and report size of all files in and below c:\temp with verbose output)
Command Line Example#2: dirZapper c:\temp -s -t (Scan and report size of all files in and below c:\temp with verbose output)
Command Line Example #3: dirZapper c:\temp -d -v (Scan and DELETE all files in and below c:\temp with verbose output)
Command Line Example #4: dirZapper c:\temp -d -t (Scan and DELETE all files in and below c:\temp with terse output)
